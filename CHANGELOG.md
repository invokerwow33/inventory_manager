# Changelog

Все значимые изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Added
- Добавлен CI/CD пайплайн для автоматизации сборки и тестирования
  - CI workflow (`ci.yml`) - анализ кода, форматирование и тесты
  - Build workflow (`build.yml`) - сборка Windows .exe, Web и Android .apk
  - Release workflow (`release.yml`) - автоматическое создание GitHub релизов
  - Deploy Web workflow (`deploy-web.yml`) - деплой на GitHub Pages
- Кэширование зависимостей Flutter для ускорения сборки

## [1.0.0+1] - 2024-XX-XX

### Added
- Полная система управления инвентарем и активами
- Система управления кинотеатром (кассы, билеты, кинозалы, сеансы)
- Управление сотрудниками и привязка оборудования
- Учет расходных материалов
- QR-коды и штрих-коды (генерация и сканирование)
- Аналитика и отчеты с графиками
- Экспорт в CSV и Excel
- Печать документов в PDF
- Синхронизация данных между устройствами
- Система задач с уведомлениями
- Многоязычная поддержка (RU/EN)
